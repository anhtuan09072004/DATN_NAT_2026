# DATN_NAT_2026 - Website bán áo phông

## 📌 Giới thiệu
Dự án bán áo phông gồm:
- Admin (quản lý sản phẩm, hóa đơn)
- Khách hàng (mua hàng online)

## ⚙️ Công nghệ
- Backend: Spring Boot
- Frontend: ReactJS
- Database: SQL Server

## 🚀 Cách chạy project

### 1. Backend
cd Backend_2026
mvn spring-boot:run

### 2. Frontend Admin
cd fe
npm install
npm run dev

### 3. Frontend Khách
cd fe_KhachHang
npm install
npm run dev

### 4. Database
Import file DATABASE_SQL.sql vào SQL Server
